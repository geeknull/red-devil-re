## 0. 全局架构速览

```
GameMIDlet (MIDlet 入口 + 字节IO原语 + 音频/存档)
  └── a   主 Canvas / 线程循环 / UI 状态机 / 菜单 / Logo / 胜负画面
        │   （持有 g y=玩家、j z=当前场景）
        ├── j   场景/关卡运行时（实例对象，非静态门面）  ← 加载 s.bin
        │        ├── b   地图瓦片层(TileMap)            ← 加载 m.bin/p.bin/b.bin + fpng/bpng
        │        ├── d   动作/动画定义(SpriteDef,静态库) ← 加载 a.bin
        │        ├── i   图块切片表/PNG贴图(TileSheet)   ← 加载 t.bin + actorPng.bin
        │        └── h   Actor 基类（坐标/速度/动画/碰撞）
        │              ├── g  玩家(PlayerActor)         （a 持有为 a.y）
        │              ├── f  普通敌人/步兵(EnemyActor)
        │              ├── e  道具/特殊单位(ItemActor)
        │              ├── k  子弹/投射物/手雷(ProjectileActor)
        │              └── c  Boss/机关/触发器(BossActor)
```

数据来源一览（详见《资源映射.md》）：
- `a.bin → d`（动作帧定义）
- `m.bin/p.bin/b.bin → b`（地图网格 / 瓦片调色 / 背景层）
- `s.bin → j`（场景：actor 类型表、触发器、actor 实例、分屏 spawn 列表）
- `t.bin → i`（图块切片表 sprite-sheet）
- `actorPng.bin/fpng.bin/bpng.bin/logo.bin → GameMIDlet.a(path,i)`（PNG 图）
- `sound.bin → GameMIDlet.b()`（2 条**标准 MIDI**）
- `string.bin → GameMIDlet.a(i,path)`（9 条 UTF-16LE 中文串）

## 与游戏1 的引擎差异点

| 维度 | 游戏1（红魔特种兵） | 游戏2（深海战舰） |
|---|---|---|
| **类数量** | 13 类（`GameMIDlet`+`a`~`l`） | **12 类**（少 `l`：子弹独立类被并入 `k`） |
| **架构重心** | `a` 巨型类（1376 行）：循环+状态机+HUD+相机+**全部关卡脚本** | **职责拆分**：`a`(1008) 只管 UI/菜单；`j`(1364) 成为**场景实例**，承载相机+剧情+spawn |
| **`j` 类性质** | 静态门面（LevelLoader） | **实例对象**（Scene），有完整生命周期与子状态机 `w` |
| **像素/贴图** | `i` 自解码 4bit packed + RGB444 调色板 + `DirectGraphics.drawPixels` | **`i` 直接 `Image.createImage(PNG)` + `drawRegion`**（标准 API，无自定义解码） |
| **图像资源分包** | 单包 `image.bin`（9 张 PNG/HUD） | **多 PNG 包**：`actorPng.bin`(9) / `fpng.bin`(2,前景) / `bpng.bin`(1,背景) / `logo.bin`(6) |
| **地图资源** | `f.bin`（地图，行编码 0/1/2）+ `t.bin`（图集+调色板） | **三件套**：`m.bin`(块网格) + `p.bin`(瓦片调色+属性条带 RLE) + `b.bin`(背景层) |
| **图块切片表** | 含在 `t.bin`（带像素+调色板） | `t.bin` 仅存 cell 坐标表（指向 actorPng 的 PNG） |
| **音频** | `sound.bin` = **6 条 WAV**（`a.g()` 加载，gain 按大小） | `sound.bin` = **2 条标准 MIDI**（`createPlayer(...,"audio/midi")`，音量 90） |
| **音频条目数** | `h`/`S[]` = 6 | `h`/`i[]` = **2** |
| **文本资源** | `x.bin`（3 条 UTF-16LE） | **`string.bin`（9 条 UTF-16LE）**：开场/对话×多关/帮助/关于 |
| **存档键** | （游戏1 各自） | RecordStore **"REDDEVIL2"**，5 字节 `{关,继续数,声音,进度,生命}` |
| **关卡数** | 8 关（0~7） | **7 关**（0~6） |
| **帧节奏** | 100ms/帧 | **80ms/帧**（更快） |
| **屏幕** | 随机型（约 176×144/176） | 固定 **176×204**（可视区 176×172 + 底部 HUD 32px） |
| **碰撞属性编码** | 0空/1实心/2可爬/3边界 | `&3 != 0`=实心、属性 `==4`=可站立平台面（CFR `g.t()`/`g.d()` 印证；其余位细分 ⚠️待核实） |
| **公共原语** | `GameMIDlet.a(String,int)` 只返回定位流；`a(IS)`=u16 | `b(String,int)` 定位流 + **`c(String,int)` 整条读取**（PNG/字符串用）；`b(IS)`=s16、`a(IS)`=byte（命名调换） |

> 共同点（同源 tjge 引擎）：MIDlet 构造 `new Random()+setSeed(currentTimeMillis())`；`GameMIDlet.a(int)` 取绝对值取模随机；`.bin` 归档容器格式完全一致；定点数 `<<10`；80ms~100ms 定帧 `run()` 循环 + `repaint()`；Actor 基类同构（坐标/速度/动作/碰撞箱/翻转位）。
